✅ 2(C) Prime Number Analysis Tool
Problem Statement

Develop a program to display all prime numbers in a range and compute count and average.

Algorithm
Input start and end
For each number, check divisibility
Print primes
Count primes
Compute average
Code
start = int(input("Enter Starting Number: "))
end = int(input("Enter Ending Number: "))

prime_list = []

for num in range(start, end + 1):
    if num > 1:
        is_prime = True
        for i in range(2, num):
            if num % i == 0:
                is_prime = False
                break
        if is_prime:
            prime_list.append(num)

print("\nPrime Numbers:", prime_list)
print("Total Prime Numbers:", len(prime_list))

if len(prime_list) > 0:
    avg = sum(prime_list) / len(prime_list)
    print(f"Average of Prime Numbers: {avg:.2f}")
else:
    print("No prime numbers found.")
Sample Output
Enter Starting Number: 10
Enter Ending Number: 30

Prime Numbers: [11, 13, 17, 19, 23, 29]
Total Prime Numbers: 6
Average of Prime Numbers: 18.67
Result

Thus the tool identifies primes, counts them, and calculates average.

✅ 6(B) Factorial-Based Password Strength Checker

Algorithm
Recursive factorial(n)
Input password length
Compute factorial
Classify Weak/Medium/Strong
Display result
Code
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

def check_strength(length):
    score = factorial(length)

    if score < 100:
        return "Weak"
    elif score <= 10000:
        return "Medium"
    else:
        return "Strong"

length = int(input("Enter password length: "))
strength = check_strength(length)

print("Password Strength:", strength)
Sample Output
Enter password length: 6
Password Strength: Medium
Result

Thus factorial recursion is used to determine password strength.

✅ 7(C) Smart City Management System (Inheritance)
Problem Statement

Develop a Smart City system demonstrating hierarchical, multilevel, and multiple inheritance.

Algorithm
Create base class Person
Create Citizen and Employee (hierarchical)
Create Engineer (multilevel)
Create SmartWorker (multiple inheritance)
Display outputs
Code
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display_person(self):
        print("Name:", self.name)
        print("Age:", self.age)

    def is_adult(self):
        return self.age >= 18

    def greet(self):
        print("Hello,", self.name)

class Citizen(Person):
    def __init__(self, name, age, city):
        super().__init__(name, age)
        self.city = city

    def show_city(self):
        print("City:", self.city)

class Employee(Person):
    def __init__(self, name, age, salary):
        super().__init__(name, age)
        self.salary = salary

    def calculate_tax(self):
        return self.salary * 0.1

class Engineer(Employee):
    def __init__(self, name, age, salary, department):
        super().__init__(name, age, salary)
        self.department = department

    def show_department(self):
        print("Department:", self.department)

class SmartWorker(Citizen, Employee):
    def __init__(self, name, age, city, salary):
        Citizen.__init__(self, name, age, city)
        self.salary = salary

    def show_role(self):
        print("Role: Citizen + Employee")

sw = SmartWorker("Arun", 25, "Chennai", 50000)
sw.display_person()
sw.show_city()
print("Tax:", sw.calculate_tax())
sw.show_role()
Sample Output
Name: Arun
Age: 25
City: Chennai
Tax: 5000.0
Role: Citizen + Employee
Result

Thus the program demonstrates hierarchical, multilevel and multiple inheritance in Python.
